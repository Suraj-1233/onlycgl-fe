SSC MTS 2024: All India Mega Live Test

Duration: 90 Mins

Read the following instructions carefully.

  1. The test contains 2 sessions having 90 questions in total. Session I is qualifying in nature and its marks will not be added to the final merit list.
  2. Each question has 4 options out of which only one is correct.
  3. You have to finish the test in 90 minutes, 45 minutes for each session.
  4. Try not to guess the answer as there is negative marking.
  5. You will be awarded 3 marks for each correct answer in both sections and no marks will be deducted in session I, and 1 negative mark for each wrong answer in session II.
  6. There is no negative marking for the questions that you have not attempted.
  7. You can write this test only once. Make sure that you complete the test before you submit it and/or close the browser.
